# Requisitos Funcionais
O sistema deve permitir que o usuário mantenha as suas transações financeiras
O sistema deve permitir que o usuário mantenha as suas despesa
O sistema deve permitir que o usuário mantenha as suas rendas
O sistema deve permitir o controle de mídias

# Requisitos Não Funcionais

# Regras de Negócio
As Transações Financeiras podem ser do tipo: Despesa ou Renda

As Transações Financeiras podem ter ocorrência do tipo: Única ou Programática

O estatus de uma Transação Financeira pode ser: Pendente; Atrasado; Cancelado; Realizado

Para cadastrar uma transação financeira, é necessário: o título da transação; descrição; data da transação; o valor; seu tipo (Despesa ou Renda); tipo de ocorrência; estatus; mídias

Para cadastrar uma transação do tipo Despesa, é necessário: nome do remetente; nome do recebedor; data limite (opcional)
